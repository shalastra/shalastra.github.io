Reading a file with Java Stream

Since Java 8 has been released, new ways of solving common issues appeared. One of them is Stream API. Except the main purpose, which are collection manipulations, there is another purpose - file reading.
But before that, let’s dive in the classic way, pre-stream world.

Old way

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Example HTML5 Document</title>
</head>
<body>
  <p>Test</p>
</body>
</html>

Stream based

<!DOCTYPE html>

Small improvement

Previous solution looks pretty good, right? Unfortunately, we have forgotten about one thing - closing the stream.
Luckily, Java provides a neat way of handling resources usage, and closing them just after we don’t need them anymore - try-with-resources.
